Animal Trading Card Work Through Points

  • Customize the Card Content
    • Add an Animal image to the card replacing the default placeholder image
      • This involves both uploading the new image file and linking to it in th HTML
      • Do not forget to add an `alt` tag to the image to provide some context for it in the case it doe snot show up, or if a screensreader is being used
    • Customize the Text on the Card to match your animal's information
  • Link the Style Sheet and Add Class Attributes
    • Link to the stylesheet in the <head> of the HTML file
      • HINT: Place the link to the stylesheet link right under the <title> element in the >head< element
    • Add class to element that you want to style
      • The main container element will be the card. Looking at the HTML, which element contains all the content for the card in it? This will be what you want ot style as the card
    • Add your CSS
      • Make sure the image has a width set to 300px
        • HINT: I would avoid setting a `height` value on the image too. Setting both height and width will likely warp the image.
        • Set a width on the card itself. Remember that the image is 300px so that card needs to take that plus any padding added to the sides into account in it's width number.
          • Example: If the image is 300px and the padding on each side is 25px, then 25px + 300px + 25px = 350px total width. Remeber is is 50 added because there is 25ps ont he left AND right side.
        • Make the text for the animal's description italic. To target this in CSS you can use the font-style property
        • Remove the bullet points from the list
          • This requires targeting the `list-style` property of the <ul> element. So add a class to the ul and then target that property to remove the bullet points.
          • Make the Parts of the list before the `:` bold.
            • You can add a class to the <span> tag and then target the `font-weight` property
          • Add a border around the animal name, image and information sections
            • first identify the container elements for each of these. Then add the border.
              • HINT: If you do not want to see a border just make the border color match the background
              • HINT: Remember, a border is not just about color, you can also round out the edges using the border-radius property.
      • Ready for submission
        • Double check the rubric here: https://review.udacity.com/#!/rubrics/3380/view
        • Make sure you code is indented correctly
        • Valid the HTML by copy and paste here: https://validator.w3.org/#validate_by_input
          • If you get errors don;t worry. The output will be massinve but focus on the highlighted parts. Ready it out loud if it helps and see if it clicks what needs to be changed. Take your time reading over it.
        • Remember you cannot use any inline styles or internal stylsheets.
          • Inline styles use `style="..."` to set properties right on HTML elements
          • An internal stylesheet is wrapped in the HTML <style> </style> tags.
        • Also make sure your class names are meaningful to what they are doing
